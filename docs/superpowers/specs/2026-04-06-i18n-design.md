# Internationalization (i18n) Design

## Overview

Add bilingual support (English/Chinese) to xhub Video Downloader using JSON translation files and a lightweight i18n module. Language is auto-detected from system locale; English is the default fallback.

## Architecture

### Files to Create

```
src/
├── i18n.py                 # Core i18n system
└── translations/
    ├── en.json             # English translations
    └── zh.json             # Chinese translations
```

### Files to Modify

- `src/main_window.py` — Wrap all UI strings with `_()` function
- `src/widgets/download_history.py` — Wrap UI strings
- `src/widgets/video_info_panel.py` — Wrap UI strings
- `src/widgets/download_progress.py` — Wrap UI strings
- `src/widgets/file_exists_dialog.py` — Wrap UI strings
- `src/widgets/actor_search_tab.py` — Wrap UI strings
- `src/config.py` — Persist language preference (optional, for future manual override)

## Implementation Details

### i18n.py Core

```python
import json
import os
from PyQt6.QtCore import QLocale

_translations = {}
_current_lang = "en"

def init_i18n():
    """Initialize i18n with system language detection."""
    global _current_lang
    system_lang = QLocale.system().bcp47Name().split("-")[0]
    _current_lang = system_lang if system_lang in ["en", "zh"] else "en"
    _load_translations()

def _load_translations():
    """Load translation JSON files."""
    global _translations
    base_dir = os.path.dirname(os.path.abspath(__file__))
    for lang in ["en", "zh"]:
        path = os.path.join(base_dir, "translations", f"{lang}.json")
        with open(path, encoding="utf-8") as f:
            _translations[lang] = json.load(f)

def _(key: str) -> str:
    """Translate a string key to the current language."""
    return _translations.get(_current_lang, {}).get(key, key)

def set_language(lang: str):
    """Manually set language (for future settings UI)."""
    global _current_lang
    if lang in _translations:
        _current_lang = lang
```

### Initialization

At the top of `main.py` or `main_window.py`:

```python
from src.i18n import init_i18n, _
init_i18n()
```

## Testing

- Run the app on a system with English locale — all text should be English
- Run the app on a system with Chinese locale — all text should be Chinese
- Any string not in the translation files will return the key itself (graceful fallback)
